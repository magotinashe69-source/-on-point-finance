Phase 1 complete — the skeleton runs

This is the secure shell of the app. There is no login screen or data yet; those arrive in the next phases. What is already wired and working:

  • Application factory — the app is built inside create_app().
  • Config from environment — the secret key is read from .env, never hard-coded.
  • Database layer (SQLAlchemy) ready for the models in Phase 2.
  • Login manager ready for the auth system in Phase 3.
  • CSRF protection active for every future form.
  • Rate limiter ready to defend the login route.
  • Hardened cookies — HttpOnly, SameSite=Lax, 30-minute auto-logout.
